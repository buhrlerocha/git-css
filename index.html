Comandos para la consola

  1. Git Init :

    2. Para crear un nuevo repositorio en git

    4. Sirve para hacer comentarios para los cambios subidos

  4. Git clone :

    5. Clona un repositorio remoto dentro de un directorio

  5. Git Status :

    6. Imprime un reporte del estado actual del árbol de trabajo local

  6. Git Help comando:

    7. Muestra la ayuda para ese comando
